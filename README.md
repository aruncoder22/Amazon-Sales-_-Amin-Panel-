# Amazon Sales & Customer Intelligence Dashboard (Power BI)

##  Project Overview

This project focuses on analyzing Amazon product sales, revenue, and customer behavior using an interactive Power BI dashboard.
The dashboard provides insights into product performance, category sales, revenue trends, and customer ratings to support data-driven decision-making.

##  Objectives

* Analyze product sales and revenue trends
* Understand category and sub-category performance
* Evaluate customer ratings and product popularity
* Provide insights through an interactive admin dashboard

##  Tools & Technologies

* Power BI
* CSV Dataset
* Data Visualization Techniques

##  Key Dashboard Features

### Sales & Revenue Overview

* Total Revenue
* Total Orders
* Revenue Trend Analysis
* Average Product Rating

### Product Performance Analysis

* Top 10 Best-Selling Products
* Product rating comparison
* Brand-wise analysis
* Category performance insights

### Admin Dashboard

* Total Revenue, Profit, Discount, and Sales
* Category and Sub-category analysis
* Total items and ratings across products
* Category-wise revenue and profit visualization

### Detailed Category Insights

* Category-specific revenue and profit
* Sub-category breakdown
* Product-level analysis
* Time-based sales trends

##  Visualizations Used

* Bar Charts
* Pie Charts
* Map Charts
* Line Charts
* Tables & KPI Cards
* Interactive Filters (Slicers)
